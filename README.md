# TuiGameCollection
Snake and IDK


## Install

### Install the newes python version: 3.10

### Create python venv

```bash
python3 -m venv <path_to_venv>
```

### Activate venv

 - Windows
  ```bash
  <path_to_venv>/Scripts/activate
  ```
- Linux
```bash
source <path_to_venv>/bin/activate
```


### Install python dependencies

```bash
python3 install -r requirements.txt
```
